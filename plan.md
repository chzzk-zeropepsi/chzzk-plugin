# 치지직 플러그인

치지직 보조 확장. 우선 3가지 기능.

## 1. 다시보기 북마크
- **생방송 중**: 현재 라이브 페이지에 "북마크" 버튼. 누르면 현재 스트림 경과 시간(초)을 채널/방송 키와 함께 저장.
- **다시보기 진입 시**: 같은 방송 키의 VOD가 열리면 저장된 북마크 목록을 사이드 패널/플레이어 위에 시간 점프 버튼으로 노출.
- **데이터**: chrome.storage.local. 키 `bookmarks/{liveId}` → `[{at: seconds, label, createdAt}]`.
- **방송 키 매핑**: 라이브 시작 시 `liveId` ↔ `videoNo`(VOD ID) 매핑이 필요. VOD 페이지 진입 시 chzzk API로 해당 VOD의 원본 라이브 정보를 조회해 매칭.

## 2. 생방송 알림 (우측 상단 토스트)
- **대상**: 사용자가 명시적으로 지정한 "알림 받을 스트리머 목록" (전체 팔로잉 X)
- **폴링**: background에서 `alarms`로 30초~1분 간격, 대상 채널의 라이브 상태 조회 (chzzk public API)
- **알림 UX**: 오프→온 전환 감지 시, 치지직 탭 어디에서나 우측 상단에 카드 표시 (썸네일, 닉네임, 카테고리, 입장 / 닫기 / 다시 보지 않기)
- **chrome.notifications**도 옵션으로 (탭 없을 때)

## 3. 팔로잉 채널 그룹화
- **그룹 정의**: 사용자가 만든 그룹(이름, 색상)에 팔로잉 채널을 끌어넣음
- **표시**: 치지직 좌측 사이드바의 팔로잉 영역을 그룹별로 묶어 렌더링 (또는 별도 오버레이 사이드바)
- **정렬**: 그룹 내 사용자 정의 순서 / 라이브 우선 / 가나다 정렬 토글
- **데이터**: `groups: [{id, name, color, channelIds: [...]}]`
