# 치지직 플러그인

![Downloads](https://img.shields.io/github/downloads/chzzk-zeropepsi/chzzk-plugin/total?label=%EB%8B%A4%EC%9A%B4%EB%A1%9C%EB%93%9C&color=1AE192&style=flat-square&cb=1)

치지직 보조 Chrome 확장.

## 기능

### 1. 팔로잉 그룹 패널 (로그인 필요)
- 우측에 떠있는 패널에서 팔로잉 채널을 **사용자 정의 그룹**으로 묶기
- **게임별 자동 분류** 탭: 현재 라이브 중인 채널을 카테고리별 자동 그룹핑
- 라이브/오프라인 표시, 시청자 수, 라이브 타이틀 툴팁
- **채널 검색**, 드래그&드롭으로 그룹 이동, 호버 시 알림 토글/그룹 제거/팔로우 취소
- 패널 드래그 이동, 리사이즈, 핀(위치 고정)
- 그룹 헤더 클릭으로 접기/펼치기

### 2. 생방송 알림
- 팝업의 "알림" 탭에서 알림 받을 스트리머 선택 (전체 팔로잉 노이즈 제거)
- 1분 주기 폴링으로 off→on 전환 감지
- 치지직 탭이 열려있으면 우상단 토스트 표시 (썸네일, 닉네임, 라이브 타이틀, 카테고리)
- "방송 보러가기" / "닫기" 누르기 전까지 유지

### 3. 다시보기 북마크
- 라이브 중 패널 헤더의 **🔖 북마크** 버튼으로 현재 시점 저장 (라벨 입력 가능)
- 해당 방송 다시보기 페이지 진입 시 좌측에 북마크 패널 자동 표시
- 시간 클릭 → 비디오 해당 시점으로 점프
- 팝업 "북마크" 탭에서 전체 북마크 관리 (라벨 인라인 편집, 삭제)

### 4. 클립 다운로드
- 단일 클립 페이지: 패널 헤더의 **🎬 다운로드** 버튼
- 채널 클립 목록 페이지: 각 썸네일 호버 시 **⬇** 오버레이
- mp4 직접 다운로드, 무한 스크롤 페이지네이션도 자동 처리

### 5. 세로 레이아웃 (긴 화면)
- 라이브 페이지 플레이어 하단의 **긴 화면** 버튼으로 영상↑/채팅↓ 세로 배치 전환
- 영상/채팅 사이 경계 드래그로 비율 조절 (위치 저장)
- 진입 시 사이드바 자동 접기 → 영상 폭 확보
- 후원 랭킹/방송 상세정보/푸터 자동 숨김

### 6. 헤더 툴바 버튼
- 치지직 상단 헤더 **스튜디오** 버튼 옆에 플러그인 로고 버튼 추가
- 클릭 시 팔로잉 패널 표시/숨김 토글

### 7. 기능별 ON/OFF
- 팝업 상단에서 팔로잉 패널 / 세로 레이아웃 / 다시보기 북마크 / 다운로드 버튼 기능을 개별 토글
- 변경 시 열린 치지직 탭 자동 새로고침

### 8. 다시보기 다운로드
- 채널 다시보기 목록 페이지: 썸네일 호버 시 **⬇** 오버레이
- **화질 선택 모달**: 1080p/720p/.../144p, 화질별 예상 용량 표시
- 동시 8개 segment 병렬 다운로드 + 진행률 바
- `.ts` 형식으로 저장 (VLC/PotPlayer 등에서 재생, ffmpeg으로 mp4 변환 가능)

## 설치

1. zip 다운로드 → 압축 해제
2. Chrome 주소창에 `chrome://extensions/` 입력
3. 우상단 **개발자 모드** 토글 ON
4. **압축해제된 확장 프로그램 로드** 클릭 → 푼 폴더 선택
5. 치지직에서 사용

## 주의

- 팔로잉 그룹 패널은 치지직 로그인이 필요합니다
- 신규 다시보기는 트랜스코딩 처리 완료(수십분~수시간 후)까지 다운로드 불가
- 매우 긴 다시보기(>1시간 1080p)는 GB 단위라 메모리 사용 큼 — 모달의 예상 용량 보고 화질 선택
- 모든 데이터는 `chrome.storage.local`에 로컬 저장 (확장 제거 시 삭제)
